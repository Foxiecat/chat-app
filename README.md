# Chat App
1. *What is the project?*
The chat app project is a project to build a simple and light chatting application. It will allow users to create an account, then make their own communities with channels that one can chat in with friends. It will also allow users to directly message other users. The project is going to be solved using:
- **API (Might change)**
    - Asp.Net Core
    - PostgreSQL
- **Client (This might change)**
    - React

2. *What is the MVP?*
The Minimal Viable Product is a chat application that allows the creation of a User, Communities and Channels within the Communities. Then allows the logged in user to chat with people either directly, or in Communities they are apart of.

3. *What are the nice to haves?*
The nice to haves is a styled application and/or website, the ability to customize User, Communities and Channels beyond what is necessary for them to work.

4. *When will the project be complete?*
The project will be complete once all the MVP features have been implemented and the app has been minimally styled, afterwards I might also implement the Nice To Haves.
